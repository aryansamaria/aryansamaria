# Aryan Samaria - Portfolio Website

A modern, high-performance personal portfolio website showcasing my experience, projects, and skills as a Python Developer and AI Engineer.

## 🚀 Features

- **Modern Design**: Clean, professional design with smooth animations and transitions
- **Fully Responsive**: Mobile-first approach ensuring perfect display on all devices
- **Performance Optimized**: Fast loading times and smooth scroll animations
- **Accessibility**: Semantic HTML, proper ARIA labels, and keyboard navigation support
- **SEO Friendly**: Proper meta tags and structured content

## 📁 Project Structure

```
Portfolio/
├── index.html          # Main HTML file
├── styles/
│   └── main.css       # All styles and animations
├── scripts/
│   └── main.js        # JavaScript for interactions and animations
├── assets/            # Images and other assets (if needed)
├── latex.tex          # Resume source (LaTeX)
├── webelemt.txt       # Reference website patterns
└── README.md          # This file
```

## 🎨 Design Features

### Sections
- **Hero**: Eye-catching introduction with gradient text and social links
- **About**: Personal introduction and background
- **Experience**: Timeline-based work experience display
- **Projects**: Card-based project showcase
- **Skills**: Categorized skill tags with hover effects
- **Education**: Academic background
- **Achievements**: Tech fest and co-curricular accomplishments
- **Contact**: Contact information and social links

### Animations
- Smooth scroll-triggered section reveals
- Hover micro-interactions on cards and buttons
- Parallax effects (optional, commented out)
- Mobile-friendly navigation menu

## 🛠️ Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with CSS variables, flexbox, and grid
- **JavaScript (ES6+)**: Vanilla JS for interactions and animations
- **Google Fonts**: Inter and Outfit font families

## 📱 Responsive Breakpoints

- **Desktop**: 1200px and above
- **Tablet**: 768px - 1199px
- **Mobile**: Below 768px

## 🚀 Getting Started

1. Clone or download this repository
2. Open `index.html` in a modern web browser
3. No build process required - it's pure HTML, CSS, and JavaScript!

## 🌐 Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## 📝 Content Source

All content is extracted from `latex.tex` (the resume LaTeX file). The website serves as a visual representation of the resume with modern UI/UX patterns inspired by the reference website.

## 🎯 Key Features Implementation

### Navigation
- Fixed navbar with scroll effects
- Smooth scrolling to sections
- Active link highlighting
- Mobile hamburger menu

### Animations
- Intersection Observer API for scroll-triggered animations
- CSS transitions for smooth interactions
- Reduced motion support for accessibility

### Performance
- Lazy loading support for images
- Optimized CSS with variables
- Minimal JavaScript footprint
